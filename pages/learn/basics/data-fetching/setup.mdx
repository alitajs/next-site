### 下载入门代码(可选)

如果你不是从上一节课继续，你可以下载、安装并运行下面这节课的入门代码。这样就建立了一个 `nextjs-blog` 目录，使其与前一节课的产物相同。

同样，如果你刚刚上完了前一课，这就没有必要了。

```shell
npx create-next-app nextjs-blog --use-npm --example "https://github.com/vercel/next-learn-starter/tree/master/data-fetching-starter"
```

然后按照命令输出中的指令(`cd` 进入目录并启动开发服务器)。

你还应该更新以下文件:

- `public/images/profile.jpg` 是你的照片(建议: 400px width/height)。
- 在 `components/layout.js` 中的 `const name = '[Your Name]'` 是你的名字。
- 在 `pages/index.js` 中的 `<p>[Your Self Introduction]</p>` 是你的自我介绍.
