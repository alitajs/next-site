到目前为止，我们创建的 Next.js 应用程序只有一个页面。而通常一个网站或 web 应用会有很多不同的页面。

接下来我们就开始探索如何为我们的应用添加多个页面。

## 你将从这节课中学到什么

在这一课，你将:

- 利用文件系统路由的功能创建一个新页面。
- 学习如何通过 `Link` 组件实现客户端页面间的导航。
- 学习内置的代码分割和预取功能。
