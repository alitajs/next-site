在 Next.js 中，页面是一个从 [`pages` 目录](/docs/basic-features/pages)中的文件导出的 React 组件。

页面与基于**文件名**的路由对应，例如，在开发中:

- `pages/index.js` 对应路由 `/`。
- `pages/posts/first-post.js` 对应路由 `/posts/first-post` 。

我们已经有了 `pages/index.js` 文件，那么让我们创建 `pages/posts/first-post.js`，看看它是如何运行的。

### 创建一个新的页面

在 `pages` 目录下新建 `posts` 目录。

在 `posts` 目录下新建一个以 `first-post.js` 命名的文件，内容如下:

```jsx
export default function FirstPost() {
  return <h1>First Post</h1>
}
```

组件可以有任何名称，但必须将其导出为 `default` 。

现在，确保开发服务器正在运行并访问 [http://localhost:3000/posts/first-post](http://localhost:3000/posts/first-post)。你应该看到这个页面:

<Image
  shadow
  alt="First Post"
  oversize={false}
  src="/static/images/learn/navigate-between-pages/first-post.png"
  width={656 / 2}
  height={282 / 2}
/>

这就是在 Next.js 中创建不同页面的方法。

只需在 ['pages' 目录](/docs/basic-features/pages)下创建一个 JS 文件，该文件的路径就成为 URL 路径。

在某种程度上，这类似于使用 HTML 或 PHP 文件。你编写 JSX 和使用 React 组件来替代写 HTML。

让我们添加一个链接到新添加的页面，以便可以从主页导航到它。

export default ({ children }) => <Layout meta={meta}>{children}</Layout>
