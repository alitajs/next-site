现在让我们谈谈 **CSS 样式**。

如你所见，我们的首页 ([http://localhost:3000](http://localhost:3000)) 已经有一些样式。如果你查看 `pages/index.js`，你应该看到如下代码:

```jsx
<style jsx>{`
  …
`}</style>
```

这是使用一个名为 [styled-jsx](https://github.com/vercel/styled-jsx) 的库。 它是一个 “CSS-in-JS” 的库 — 它允许你在一个 React 组件中编写 CSS，并且 CSS 样式将被限定在_当前范围_内(其他组件不会受到影响)。

Next.js 内置支持了 [styled-jsx](https://github.com/vercel/styled-jsx)，但是你也可以使用其他流行的 CSS-in-JS 库，比如 [styled-components](https://github.com/vercel/next.js/tree/canary/examples/with-styled-components) 或者 [emotion](https://github.com/vercel/next.js/tree/canary/examples/with-emotion)。

### 编写和导入 CSS

Next.js 内置支持 CSS 和 Sass的支持，允许你导入 `.css` 和 `.scss` 文件 。

也支持使用流行的 CSS 库，比如 [Tailwind CSS](https://github.com/vercel/next.js/tree/canary/examples/with-tailwindcss)。

在本课中，我们将讨论如何在 Next.js 中编写和导入 CSS 文件。我们还将讨论 Next.js 对 [CSS Modules](https://github.com/css-modules/css-modules) 和 [Sass](https://sass-lang.com/) 的内置支持。我们开始吧！
