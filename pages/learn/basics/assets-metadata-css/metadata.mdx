如果我们想修改页面的元数据，比如 `<title>` HTML 标签？

`<title>`是`<head>` 标签的一部分内容。让我们深入了解如何修改在 Next.js 页面中的 `<head>` 标签。

在编辑器中打开 `pages/index.js` 并查看以下几行:

```jsx
<Head>
  <title>Create Next App</title>
  <link rel="icon" href="/favicon.ico" />
</Head>
```

请注意，使用 `<Head>` 代替小写字母`<head>`。`<Head>` 是内置于 Next.js 中的 React 组件。它允许你修改页面的 `<head>`。

你可以从 `next/head` 模块导入 `Head` 组件。You can import the `Head` component from the `next/head` module.

### 添加 `<Head>` 到 `first-post.js`

我们还没有在我们的 `/posts/first-post` 上增加一个`<title>`，让我们添加一个。

打开 `pages/posts/first-post.js` 文件。

首先，从 `next/head` 导入 `Head`:

```js
import Head from 'next/head'
```

然后，将其添加到 `FirstPost` 组件中。现在，我们只添加 `title` 标签:

```jsx
export default function FirstPost() {
  return (
    <>
      <Head>
        <title>First Post</title>
      </Head>
      …
    </>
  )
}
```

尝试访问 [http://localhost:3000/posts/first-post](http://localhost:3000/posts/first-post)。浏览器标题现在应该是 “First Post”。通过使用浏览器的开发工具，您应该会看到 `title` 标签被添加到 `<head>` 中。
