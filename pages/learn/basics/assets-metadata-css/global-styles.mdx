CSS Modules 对于组件级样式很有用。但是如果你想加载一些 CSS 来作用于**每一页**，Next.js也支持这种方式。

要加载全局 CSS 文件，在 `pages` 下**创建一个名为`_app.js`的文件**，加上以下内容：

```jsx
export default function App({ Component, pageProps }) {
  return <Component {...pageProps} />
}
```

这个 `App` 组件是顶级组件，在所有不同的页面上都可见。例如，当在页面之间导航时，你可以使用这个 `App` 组件来保持状态。

### 重新启动开发服务器

**重要信息:** 添加 `_app.js` 时，你需要重新启动开发服务器。按 <kbd>Ctrl + c</kbd> 来停止服务器并重新运行:

```shell
npm run dev
```

### 添加全局样式

在 Next.js 中，你可以通过从 `_app.js` 导入全局样式文件来添加它们。你**无法**在其他地方导入全局样式。

全局样式不能从外部导入的原因是 `_app.js` 是全局的，它的样式影响页面上的所有元素。

如果从主页导航到 `/posts/first-post` 页面，主页的全局样式会无意中影响 `/posts/first-post`。

你可以在任何地方放置全局样式文件并使用任何名称，因此让我们这样做：

- 创建一个顶级 `styles` 目录，并在里面创建 `global.css` 文件.
- 添加下列内容。它重置一些样式并更改 `a` 标签的颜色。

```css
html,
body {
  padding: 0;
  margin: 0;
  font-family: -apple-system, BlinkMacSystemFont, Segoe UI, Roboto, Oxygen, Ubuntu,
    Cantarell, Fira Sans, Droid Sans, Helvetica Neue, sans-serif;
  line-height: 1.6;
  font-size: 18px;
}

* {
  box-sizing: border-box;
}

a {
  color: #0070f3;
  text-decoration: none;
}

a:hover {
  text-decoration: underline;
}

img {
  max-width: 100%;
  display: block;
}
```

最后，从 `_app.js` 导入它:

```jsx
import '../styles/global.css'

export default function App({ Component, pageProps }) {
  return <Component {...pageProps} />
}
```

现在，如果你访问[http://localhost:3000/posts/first-post](http://localhost:3000/posts/first-post)，你将看到样式生效了：

总结一下到目前为止我们所学到的:

- 要使用 CSS Modules，请从任何组件导入一个名为 `*.module.css` 的 CSS 文件。
- 要使用全局样式，请在 `pages/_app.js` 中导入一个 CSS 文件。
