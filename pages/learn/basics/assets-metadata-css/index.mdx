Next.js 内置了对 [CSS](/docs/basic-features/built-in-css-support) 和 [Sass](/docs/basic-features/built-in-css-support#sass-support) 的支持，在本课程中，我们将使用CSS。

本课还将讨论 Next.js 如何处理图像之类的静态资产，以及`title` 标签之类的页面元数据。

### 这节课你将学到什么

在这一课中，你会学到:

- 如何向 Next.js 添[静态文件](/docs/basic-features/static-file-serving)(图像等)。
- 如何定制每个页面的 `<head>` 内的内容。
- -如何使用 [CSS Modules](/docs/basic-features/built-in-css-support#adding-component-level-css) 创建可重用的React组件
- 如何在 [`pages/_app.js`](/docs/advanced-features/custom-app) 中添加全局 CSS。
- 在 Next.js 中使用一些实用的样式设计技巧。
