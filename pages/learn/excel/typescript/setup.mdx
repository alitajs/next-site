### 下载入门代码(可选)

If you’re NOT continuing from the basics lessons, you can download, install, and run the starter code for this lesson below. This sets up a `nextjs-blog` directory such that it’s identical to the result of the basics lessons.

Again, this is NOT necessary if you’ve just finished the basics lessons.

```shell
npx create-next-app nextjs-blog --use-npm --example "https://github.com/vercel/next-learn-starter/tree/master/basics-final"
```

然后按照命令输出中的指令(`cd` 进入目录并启动开发服务器)。

你还应该更新以下文件:

- `public/images/profile.jpg` 是你的照片(建议: 400px width/height)。
- 在 `components/layout.js` 中的 `const name = '[Your Name]'` 是你的名字。
- 在 `pages/index.js` 中的 `<p>[Your Self Introduction]</p>` 是你的自我介绍.
